\section{Methodology}
\label{sec:method}

\subsection{Problem Formulation}

Given a pre-trained multi-view 3D reconstruction model $\mathcal{M}$ and a set of test images $\{I_1, I_2, \ldots, I_N\}$ from a novel domain, our goal is to adapt $\mathcal{M}$ to improve reconstruction quality without access to ground-truth depth or camera poses. We formulate this as a self-supervised learning problem where supervision signals are derived from the internal representations of the model itself. Specifically, we leverage the observation that processing more views provides richer geometric constraints, and use this asymmetry to create a natural teacher-student distillation framework.

\subsection{Framework Overview}

Our framework consists of two networks sharing the same architecture: a frozen teacher $\mathcal{T}$ processing $N$ views and a student $\mathcal{S}$ with trainable Low-Rank Adaptation (LoRA) modules processing $M < N$ views. The teacher network, having access to more views, produces feature representations that encode richer multi-view geometric information. The student network learns to approximate these representations despite observing fewer views, effectively learning to hallucinate the geometric understanding that would come from additional viewpoints. During training, only the LoRA parameters in the student network are updated, preserving the pre-trained knowledge while enabling efficient domain adaptation.

\subsection{Low-Rank Adaptation Integration}

We integrate LoRA modules into the transformer layers of the student network to enable parameter-efficient adaptation. For a pre-trained weight matrix $W_0 \in \mathbb{R}^{d \times k}$, LoRA introduces a low-rank decomposition:
\begin{equation}
W = W_0 + \Delta W = W_0 + BA,
\label{eq:lora}
\end{equation}
where $B \in \mathbb{R}^{d \times r}$ and $A \in \mathbb{R}^{r \times k}$ are the trainable low-rank matrices, and $r \ll \min(d, k)$ is the rank. The modified forward pass becomes:
\begin{equation}
h = W_0 x + \frac{\alpha}{r} BAx,
\label{eq:lora_forward}
\end{equation}
where $\alpha$ is a scaling factor that controls the magnitude of the adaptation. We apply LoRA to the attention projection matrices in transformer layers 13 through 39, which correspond to the layers where cross-view attention mechanisms operate. With rank $r=16$ and scaling factor $\alpha=16$, we achieve effective adaptation while training only approximately 0.1\% of the total model parameters.

\subsection{Feature Extraction}

The transformer encoder produces feature tokens at intermediate layers that encode both local appearance information and global geometric context. At layer $l$, the model outputs feature tensors $\mathbf{F}^{(l)} \in \mathbb{R}^{B \times S \times (P+1) \times C}$, where $B$ denotes the batch size, $S$ is the number of input views, $P$ represents the number of spatial patch tokens, and $C$ is the channel dimension. The additional token beyond the $P$ spatial patches corresponds to the CLS token that aggregates global information across each view. In our architecture based on the Giant variant, the channel dimension is $C = 3072$, which results from concatenating two complementary feature streams: the first 1536 channels encode local per-patch information related to depth and surface properties, while the second 1536 channels encode global information including camera pose and scene-level geometry.

For distillation, we extract features from layer 39, which serves as the input to the camera pose decoder and contains the most refined multi-view representations. Given teacher features $\mathbf{F}^{\mathcal{T}} \in \mathbb{R}^{B \times N \times (P+1) \times C}$ from $N$ views and student features $\mathbf{F}^{\mathcal{S}} \in \mathbb{R}^{B \times M \times (P+1) \times C}$ from $M$ views, we first select the corresponding teacher features at the student view indices to obtain $\mathbf{F}^{\mathcal{T}}_{\text{sel}} \in \mathbb{R}^{B \times M \times (P+1) \times C}$. Each token $\mathbf{f}_{b,s,p} \in \mathbb{R}^{C}$ at batch index $b$, view $s$, and spatial position $p$ represents a unified encoding that jointly captures local depth cues and global geometric context. Rather than treating the two channel halves separately, we operate on the full concatenated representation to allow gradients to flow jointly across both components during training.

\subsection{Self-Supervised Distillation Loss}

Our distillation objective consists of two complementary components. The corresponding-frame loss aligns teacher and student features at the $M$ shared view positions on a per-token basis. The cross-frame loss transfers relational knowledge from the teacher's extra viewpoints that the student never observes. We describe each component below.

Let $\mathbf{F}^{\mathcal{T}} \in \mathbb{R}^{B \times N \times (P+1) \times C}$ denote the teacher features from all $N$ views at layer $l$, and $\mathbf{F}^{\mathcal{S}} \in \mathbb{R}^{B \times M \times (P+1) \times C}$ the student features from the $M$ shared views, where $B$ is the batch size, $P$ the number of spatial patch tokens, and $C$ the channel dimension. We select the teacher features at the student view indices to obtain $\mathbf{F}^{\mathcal{T}}_{\text{sel}} \in \mathbb{R}^{B \times M \times (P+1) \times C}$. For a given batch element, view $s$, and spatial position $p$, we write the teacher token as $\mathbf{f}^{\mathcal{T}}_{s,p} \in \mathbb{R}^{C}$ and the student token as $\mathbf{f}^{\mathcal{S}}_{s,p} \in \mathbb{R}^{C}$.

\subsubsection{Corresponding-Frame Loss}

The corresponding-frame loss operates on matched token pairs $(\mathbf{f}^{\mathcal{T}}_{s,p},\, \mathbf{f}^{\mathcal{S}}_{s,p})$ at the same view $s$ and position $p$, combining a KL divergence term for distributional alignment with a cosine similarity term for directional alignment.

\paragraph{KL divergence term.}
We apply a channel-wise softmax to convert each token into a probability distribution. For a token $\mathbf{f} \in \mathbb{R}^{C}$ the normalized representation is:
\begin{equation}
\hat{f}_c = \frac{\exp(f_c)}{\sum_{j=1}^{C} \exp(f_j)},
\label{eq:softmax}
\end{equation}
where $f_c$ is the $c$-th channel. Writing $\hat{\mathbf{f}}^{\mathcal{T}}_{s,p}$ and $\hat{\mathbf{f}}^{\mathcal{S}}_{s,p}$ for the softmax-normalized teacher and student tokens, the KL divergence loss is:
\begin{equation}
\mathcal{L}_{\text{KL}} = \frac{1}{BM(P+1)} \sum_{b,s,p} \sum_{c=1}^{C} \hat{f}^{\mathcal{T}}_{b,s,p,c} \left( \log \hat{f}^{\mathcal{T}}_{b,s,p,c} - \log \hat{f}^{\mathcal{S}}_{b,s,p,c} \right),
\label{eq:kl_loss}
\end{equation}
where the outer sum runs over batch index $b \in [1, B]$, view $s \in [1, M]$, and token position $p \in [0, P]$. This term encourages the student to reproduce the relative channel activation magnitudes of the teacher.

\paragraph{Cosine similarity term.}
We L2-normalize each token along the channel dimension to obtain unit vectors $\bar{\mathbf{f}}^{\mathcal{T}}_{s,p} = \mathbf{f}^{\mathcal{T}}_{s,p} / \|\mathbf{f}^{\mathcal{T}}_{s,p}\|_2$ and $\bar{\mathbf{f}}^{\mathcal{S}}_{s,p} = \mathbf{f}^{\mathcal{S}}_{s,p} / \|\mathbf{f}^{\mathcal{S}}_{s,p}\|_2$. The cosine loss is:
\begin{equation}
\mathcal{L}_{\text{cos}} = 1 - \frac{1}{BM(P+1)} \sum_{b,s,p} \left( \bar{\mathbf{f}}^{\mathcal{T}}_{b,s,p} \cdot \bar{\mathbf{f}}^{\mathcal{S}}_{b,s,p} \right).
\label{eq:cos_loss}
\end{equation}
This term directly maximizes the directional alignment between corresponding teacher and student tokens regardless of their magnitudes.

\subsubsection{Cross-Frame Relational Distillation}

The corresponding-frame loss aligns features at shared view positions but does not capture the geometric relationships that the teacher encodes from its additional viewpoints. To transfer this structural knowledge we introduce two cross-frame losses, one angle-wise and one distance-wise, that operate on triplets formed across the teacher's frame set.

We partition the teacher's $N$ frames into three roles according to the student's view indices. The first student frame (teacher index 0) serves as the \emph{reference} frame. The remaining student frames (teacher indices 2, 4, 6) are \emph{shared} frames visible to both networks. The teacher-only frames (indices 1, 3, 5, 7) are \emph{extra} frames that the student never observes. For computational efficiency we randomly sample $N_r$ reference patches from the reference frame and $N_s$ shared patches from each shared frame. Let $\mathbf{r}^{\mathcal{T}} \in \mathbb{R}^{C}$ and $\mathbf{r}^{\mathcal{S}} \in \mathbb{R}^{C}$ denote a sampled reference patch in the teacher and student representations respectively, and $\mathbf{s}^{\mathcal{T}} \in \mathbb{R}^{C}$, $\mathbf{s}^{\mathcal{S}} \in \mathbb{R}^{C}$ a sampled shared-frame patch. For each teacher reference patch $\mathbf{r}^{\mathcal{T}}$ we compute cosine similarity against all patches across the extra frames and select the top-$K$ most similar patches $\{\mathbf{h}_1, \ldots, \mathbf{h}_K\} \subset \mathbb{R}^{C}$. These high-similarity patches encode the geometric information that the teacher has extracted from the views unavailable to the student. The top-$K$ search and all teacher-side quantities are computed without gradients so that the losses only backpropagate through the student's feature representations. In our experiments we set $N_r = N_s = 256$ and $K = 4$, and process the computation in chunks to bound peak memory usage.

\paragraph{Angle-wise loss.}
For each triplet we compute three angles defined by the cosine of the angle at each vertex of the triangle formed in feature space. Using the teacher triplet $(\mathbf{r}^{\mathcal{T}}, \mathbf{s}^{\mathcal{T}}, \mathbf{h})$ and the student triplet $(\mathbf{r}^{\mathcal{S}}, \mathbf{s}^{\mathcal{S}}, \mathbf{h})$, the three angles are:
\begin{align}
\psi_1(\mathbf{r}, \mathbf{s}, \mathbf{h}) &= \cos\angle(\mathbf{s} - \mathbf{r},\; \mathbf{h} - \mathbf{r}), \label{eq:angle1} \\
\psi_2(\mathbf{r}, \mathbf{s}, \mathbf{h}) &= \cos\angle(\mathbf{r} - \mathbf{h},\; \mathbf{s} - \mathbf{h}), \label{eq:angle2} \\
\psi_3(\mathbf{r}, \mathbf{s}, \mathbf{h}) &= \cos\angle(\mathbf{r} - \mathbf{s},\; \mathbf{h} - \mathbf{s}), \label{eq:angle3}
\end{align}
where $\cos\angle(\mathbf{a}, \mathbf{b}) = \frac{\mathbf{a} \cdot \mathbf{b}}{\|\mathbf{a}\| \|\mathbf{b}\|}$. Together these three angles fully characterize the shape of the triangle. The student is trained to match the teacher's angular structure by minimizing the L1 distance between corresponding angles:
\begin{equation}
\mathcal{L}_{\text{CF-angle}} = \frac{1}{|\mathcal{P}|} \sum_{(\mathbf{r}, \mathbf{s}, \mathbf{h}) \in \mathcal{P}} \sum_{i=1}^{3} \omega_i \left|\psi_i^{\mathcal{S}} - \psi_i^{\mathcal{T}}\right|,
\label{eq:cf_angle_loss}
\end{equation}
where $\mathcal{P}$ is the set of all sampled triplets, $\psi_i^{\mathcal{T}}$ and $\psi_i^{\mathcal{S}}$ are the teacher and student angles respectively, and $\omega_i$ are per-angle weights set to 1 for all three terms. Because the angles are scale-invariant this loss captures the relational geometry of the triangle without being affected by the absolute magnitude of the feature vectors.

\paragraph{Distance-wise loss.}
The angle-wise loss preserves the shape of each triplet triangle but is invariant to its scale. To also transfer the scale information we add a complementary distance-wise loss that operates on the three edges of the same triplet. Rather than matching raw Euclidean distances, which can be noisy in high-dimensional spaces, we convert each pairwise difference vector into a distribution via a temperature-scaled softmax and then measure the KL divergence between the teacher and student distributions. For an edge between two points $\mathbf{a}$ and $\mathbf{b}$, the difference profile is $\mathbf{d} = \mathbf{a} - \mathbf{b} \in \mathbb{R}^{C}$ and we form the distribution $\hat{\mathbf{d}} = \text{softmax}(\mathbf{d}/\tau)$ where $\tau$ is a temperature parameter. The three edges of the triplet give rise to three KL terms:
\begin{align}
\delta_1 &= D_{\text{KL}}\!\left(\hat{\mathbf{d}}^{\mathcal{T}}_{\mathbf{r}\mathbf{s}} \;\|\; \hat{\mathbf{d}}^{\mathcal{S}}_{\mathbf{r}\mathbf{s}}\right), \label{eq:dist_d1} \\
\delta_2 &= D_{\text{KL}}\!\left(\hat{\mathbf{d}}^{\mathcal{T}}_{\mathbf{r}\mathbf{h}} \;\|\; \hat{\mathbf{d}}^{\mathcal{S}}_{\mathbf{r}\mathbf{h}}\right), \label{eq:dist_d2} \\
\delta_3 &= D_{\text{KL}}\!\left(\hat{\mathbf{d}}^{\mathcal{T}}_{\mathbf{s}\mathbf{h}} \;\|\; \hat{\mathbf{d}}^{\mathcal{S}}_{\mathbf{s}\mathbf{h}}\right), \label{eq:dist_d3}
\end{align}
where $\hat{\mathbf{d}}^{\mathcal{T}}_{\mathbf{r}\mathbf{s}} = \text{softmax}((\mathbf{r}^{\mathcal{T}} - \mathbf{s}^{\mathcal{T}})/\tau)$ and $\hat{\mathbf{d}}^{\mathcal{S}}_{\mathbf{r}\mathbf{s}} = \text{softmax}((\mathbf{r}^{\mathcal{S}} - \mathbf{s}^{\mathcal{S}})/\tau)$, and likewise for the other two edges. Each per-triplet KL value is passed through a Huber function ($\beta = 0.5$) to dampen outliers, and the loss is averaged over all triplets:
\begin{equation}
\mathcal{L}_{\text{CF-dist}} = \frac{1}{|\mathcal{P}|} \sum_{(\mathbf{r}, \mathbf{s}, \mathbf{h}) \in \mathcal{P}} \sum_{j=1}^{3} \mu_j \cdot \ell_{\beta}(\delta_j),
\label{eq:cf_dist_loss}
\end{equation}
where $\ell_{\beta}$ is the Huber loss with threshold $\beta = 0.5$ and $\mu_j$ are per-edge weights set to 1. Combined with the angle-wise loss, the distance-wise loss ensures that the student reproduces both the shape and the scale of the relational triangles that the teacher forms with its extra-frame features.

\subsubsection{Combined Loss Function}

The total distillation loss combines the corresponding-frame alignment terms with the two cross-frame relational terms:
\begin{equation}
\mathcal{L}_{\text{total}} = \lambda_{\text{KL}} \mathcal{L}_{\text{KL}} + \lambda_{\text{cos}} \mathcal{L}_{\text{cos}} + \lambda_{\text{angle}} \mathcal{L}_{\text{CF-angle}} + \lambda_{\text{dist}} \mathcal{L}_{\text{CF-dist}},
\label{eq:total_loss}
\end{equation}
where $\lambda_{\text{KL}} = 1$, $\lambda_{\text{cos}} = 2$, $\lambda_{\text{angle}} = 2$, and $\lambda_{\text{dist}} = 1$ in our experiments. The first two terms align the student's per-token features to the teacher at corresponding view positions. The cross-frame angle and distance terms complement these by transferring relational knowledge from the teacher's extra frames, encouraging the student to preserve both the directional geometry and the metric structure that the teacher encodes from viewpoints the student does not observe.

\subsection{Training Procedure}

We sample $N=8$ views for the teacher and select $M=4$ views for the student by taking every second frame (indices 0, 2, 4, 6), ensuring that the student views form a proper subset of the teacher views. To generate sufficient training samples from a limited number of scenes, we use multiple random seeds to sample different view sequences from each scene, resulting in approximately 60 training samples across the dataset. This sampling strategy maintains temporal diversity while creating a consistent correspondence between teacher and student observations.

\textbf{LoRA Configuration.} We use the PEFT library to apply Low-Rank Adaptation to the transformer backbone. LoRA modules are inserted into the attention projection matrices (qkv and output projections) and MLP layers (w12 and w3 for SwiGLU activation) in layers 13 through 39. Following the PEFT default initialization, the down-projection matrix $A$ is initialized from a normal distribution with variance $1/r$ where $r$ is the rank, while the up-projection matrix $B$ is initialized to zero, ensuring that the LoRA contribution starts at zero and gradually learns the adaptation. With rank $r=16$ and scaling factor $\alpha=16$, the effective scaling is $\alpha/r = 1$, meaning the LoRA updates are applied without additional scaling.

\textbf{Optimization Details.} We use the AdamW optimizer with a learning rate of $10^{-4}$ and weight decay of $10^{-5}$. Training proceeds for 1 epoch with a batch size of 1, without learning rate scheduling. Mixed precision training with FP16 is employed to reduce memory consumption and accelerate computation. Gradient clipping with a maximum norm of 1.0 prevents training instabilities that may arise from the KL divergence term when student predictions diverge significantly from the teacher.

\textbf{View Sampling Strategy.} For each training scene, we sample views using multiple random seeds (e.g., seeds 39, 40, 41) to generate diverse training samples from the same scene. With 3 seeds per scene and approximately 20 scenes in a typical benchmark dataset, this yields roughly 60 training samples per epoch. The first frame in each sampled sequence serves as the reference view for maintaining geometric consistency. Student views are deterministically selected as every second frame from the teacher sequence, which provides sufficient baseline separation for stereo matching while maintaining overlap for feature correspondence.
